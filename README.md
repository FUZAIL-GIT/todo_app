# APIER-CLIENT

## Tech Stack

- Framework : Flutter
- State management : Getx
- Architecture : CLEAN ARCHITECTURE
- Navigation : Go Router

## Run Locally

Clone the project

```bash
  git clone https://github.com/FUZAIL-GIT/todo_app
```

Go to the project directory

```bash
  cd todo_app
```

Install dependencies

```bash
  flutter pub get
```

Run the application on web

```bash
  flutter run
```
